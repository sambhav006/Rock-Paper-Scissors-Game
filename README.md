# Rock Paper Scissors Game

An interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features

* Real-time score tracking
* Computer-generated moves
* Battle animations
* Winner and loser visual effects
* First-to-5 win system
* Victory popup
* Confetti celebration for player wins
* Restart game functionality
* Responsive and clean UI

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Play

1. Choose Rock, Paper, or Scissors.
2. The computer randomly selects a move.
3. First player to reach 5 points wins.
4. Use the Restart button to start over anytime.
